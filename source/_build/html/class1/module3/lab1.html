Lab 3.1 - Run attacks on the API Server¶

Note

Objective: To successfully send attack traffic to API Server

Note

Prerequisite: Completion of NGINX Lab

1. Logon to Window Jump Host with credential user/user

2. Open Postman by clicking on shortcut or Clicking the icon on Quick launch toolbar

3. Open Collection Arcadia

4. Send attack traffic to server. Double click on “Post Buy Stocks XSS Attack”

5. Validate the XSS attack in action parameter. Click Send

6. Click Send and validate the attack, goes through successfully, Check HTTP Status is 200 OK
