---------------------------------------

.. note:: **Objective:** To successfully send attack traffic to API Server

.. note:: **Prerequisite:** Completion of NGINX Lab

with credential user/user

2. Open Postman by clicking on shortcut or Clicking
the icon on Quick launch toolbar
